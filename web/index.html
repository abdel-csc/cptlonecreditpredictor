01 — What actually drives default risk

Utilization matters more than credit score.

A logistic regression trained on the existing portfolio (AUC 0.72, edging out a gradient boosting benchmark at 0.71) shows utilization rate as the strongest single predictor of default, ahead of delinquency history and credit score. Values above 1.0 increase odds of default per standard deviation; below 1.0 is protective.

02 — Why that's a problem for this program

The eligibility rule selects for the risk factor that matters most.

The proposed program targets cardholders using 60%+ of their limit, exactly the feature the model flags as the top risk driver. The result: a pool that looks safe on paper (zero delinquencies) but runs meaningfully hotter than the portfolio's genuinely low-risk tier.

The query behind this chart — run against SQLite, not pandas
SELECT
    CASE
        WHEN utilization_rate < 0.30 AND num_delinquencies_24mo = 0 THEN 'Low Risk'
        WHEN utilization_rate < 0.60 AND num_delinquencies_24mo <= 1 THEN 'Medium Risk'
        ELSE 'High Risk'
    END AS risk_tier,
    COUNT(*) AS num_customers,
    ROUND(AVG(default_12mo) * 100, 2) AS default_rate_pct,
    ROUND(AVG(income), 0) AS avg_income,
    ROUND(AVG(utilization_rate) * 100, 1) AS avg_utilization_pct
FROM portfolio
GROUP BY risk_tier
ORDER BY default_rate_pct;
